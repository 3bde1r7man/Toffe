\doxysubsection{Detailed Description}
The Cart class represents a user\textquotesingle{}s shopping cart in the Toffee application. It allows the user to add, update and delete items from the cart, and view the current contents of the cart.

This class provides functionality to send emails through a Gmail SMTP server. It allows sending emails to a specified recipient email with a verification code.

This class represents an item object with a view method to display all available items in the database, and a method to get the item ID and quantity from the user, and add the item to the user\textquotesingle{}s cart.

This class represents a user who has successfully logged in to the system. It extends the User class and adds functionality related to a logged in user, such as viewing their order history and cart.

The Order class represents an order placed by a user. It provides methods for saving an order to the database, retrieving and displaying orders for a particular user, and changing the status of the most recent order for the user.

The Toffe class provides a console interface for interacting with the Toffe application. It allows users to place orders, view items and orders, and register and login to the application.

The User class represents a user of the Toffee application. It allows for registration, login, and authentication of users. It also includes methods for generating and verifying OTPs, and for checking the strength of passwords.
