Retrieves and displays all orders for a particular user from the database. It queries the \textquotesingle{}Order\textquotesingle{} table to get the order ids and then queries the \textquotesingle{}Order\+Items\textquotesingle{} table to get the details of the items in each order. It then joins the \textquotesingle{}Item\textquotesingle{} table to get the name of each item and displays the order details to the console.
