This class provides functionality to send emails through a Gmail SMTP server. It allows sending emails to a specified recipient email with a verification code.

This class represents an item object with a view method to display all available items in the database, and a method to get the item ID and quantity from the user, and add the item to the user's cart.

The Order class represents an order placed by a user. It provides methods for saving an order to the database, retrieving and displaying orders for a particular user, and changing the status of the most recent order for the user.

The Toffe class provides a console interface for interacting with the Toffe application. It allows users to place orders, view items and orders, and register and login to the application.
